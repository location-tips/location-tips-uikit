# White label palette

Palette for white label uikit design system

<Canvas>
    <ColorPalette>
    {
    Object.entries(palette.whiteLabel.palette).map((token, i) => {
        return(
            <ColorItem
                key={i}
                title={`${token[0]} color`}
                colors={
                    Object.entries(token[1]).map((value) => {
                        return `var(${value[1].cssVar})`
                    })
                }
            />
            )
        })
    }
    </ColorPalette>
